# 問題
[A - Getting Difference](https://atcoder.jp/contests/agc018/tasks/agc018_a)

# 考察

ある整数\\(g\\)の倍数である整数\\(A, B\\)どうしの差の絶対値は\\(g\\)の倍数である。

\\(|A - B| = |gp - gq| = g|p - q| \\)

好きなだけ操作を繰り返すと、最大公約数\\(G\\)を作れる。
\\(max(A_i) \geq K \\)なら0回以上\\(max(A_i)\\)から\\(G\\)を引くことで\\(K\\)を作れる。
